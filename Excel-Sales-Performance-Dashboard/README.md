# Sales Performance Analysis Dashboard

## Overview
Developed an interactive Excel dashboard to analyze sales performance across regions, products, and time periods.

## Objective
To monitor sales KPIs and identify revenue growth opportunities.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Key KPIs
- Total Revenue
- Total Profit
- Sales Growth
- Top Products
- Regional Performance

## Key Insights
- West region generated highest revenue.
- Product Category A contributed most profit.
- Q4 showed highest sales growth.
